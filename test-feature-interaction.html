Manual Verification Steps

  1. Test Simultaneous Operation:
    • Click "Enable Both"
    • Open another application (e.g., browser)
    • Verify: Window stays on top AND is hidden from taskbar
  2. Test Always-On-Top Independence:
    • Click "Enable Both"
    • Click "Toggle Always-On-Top" to disable it
    • Verify: Window can now be covered by other windows
    • Verify: Window is STILL hidden from taskbar
  3. Test Hide-From-Taskbar Independence:
    • Click "Enable Both"
    • Click "Toggle Hide-From-Taskbar" to disable it
    • Verify: Window now appears in taskbar
    • Verify: Window STILL stays on top of other windows
